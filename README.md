# Lab0_Git
Постановка:
Дан массив arr фиксированного размера 10. Требуется определить кол-во элементов, которые встречаются только один раз. Результат вывести на экран.

Описание алгоритма:
Находим максимальный элемент массива, значение которого потребуется для создание элемента "notExist". Прибавляем к нему 1, таким образом получаем элемент которого не существует в массиве.
Проходимся по массиву. Берем первый элемент и смотрим имеет ли он значение "notExist". Если да, то флаг unique = false. Сравниваем его со всеми остальными элементами. Если элементы равны, то найденный элемент заменяем на "notExist"
Если флаг unique = true, увеличиваем счетчик кол-во различных элементов. Делаем до тех пор пока не пройдемся по всем элементов. В конце выводим кол-во различных элементов
